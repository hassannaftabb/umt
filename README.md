
## Pre-Sequritiies to run on localhost:
```
NodeJS
IDE - VS Code or any.
```
## Pre-Commands:
```
npm install (in root directory(main))
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
